# MariaDB 12 for fly-NAS (fnOS)

![MariaDB Logo](ICON.PNG)

## 项目简介

**MariaDB 12** 是一款专为飞牛 **fnOS** 深度优化的原生数据库服务器应用。基于 MariaDB 12.2 (LTS) 版本构建，旨在为 NAS 用户提供高性能、低延迟且易于管理的开源关系型数据库解决方案。

本应用不仅提供了 MariaDB 的核心数据库能力，还深度集成了 fnOS 的应用中心架构，支持可视化安装向导、自定义运行端口以及系统级的自动化运维管理。

---

## 主要特性

- 🚀 **原生性能优化**：直接运行于 fnOS 系统环境，相比容器化部署具有更低的资源开销和更高的 I/O 效率。
- 🔧 **高度可定制化**：支持在安装或配置向导中自定义服务端口 (默认 3306)，轻松解决端口冲突问题。
- 🛡️ **安全安装向导**：首创引导式安装流程，强制要求设置 Root 管理员密码，确保数据库初始化即安全。
- 📊 **可视化仪表板**：集成 fnOS 桌面快捷方式，提供直观的服务状态监控与基础管理入口。
- 📂 **透明化路径管理**：
  - **安装路径**：`/usr/local/apps/@appcenter/MariaDB12`
  - **数据存储**：持久化存储于应用专有的 `var` 目录，确保系统升级不丢失数据。
  - **配置文件**：通过主程序动态生成的 `my.cnf` 进行精确控制。
- 🔄 **完善的生命周期管理**：支持一键安装、平滑升级、配置重载及彻底卸载。

## 目录结构

```text
.
├── app/                 # 应用核心程序
│   ├── target/          # MariaDB 二进制执行文件 (x86_64/aarch64)
│   ├── etc/             # 配置文件模板 (my.cnf.template)
│   └── ui/              # 桌面仪表板 UI 资源
├── cmd/                 # 飞牛扩展脚本 (生命周期管理)
│   ├── main             # 主服务控制脚本 (start|stop|status)
│   ├── install_init     # 安装初始化
│   ├── install_callback # 安装后回调 (数据库初始化及权限设置)
│   └── config_callback  # 配置修改回调 (端口/密码变更)
├── config/              # 应用静态配置
├── wizard/              # 可视化向导定义 (JSON)
│   ├── install          # 安装向导：设置端口与密码
│   └── config           # 修改配置向导
├── manifest             # 应用元数据清单
└── ICON.PNG             # 应用图标
```

## 安装与使用

1. **安装**：在 fnOS 应用中心选择 "MariaDB 12" 点击安装。
2. **初始化**：在弹出的安装向导中：
   - 输入您希望使用的 **数据库端口** (如 3310)。
   - 输入并确认您的 **Root 管理员密码**。
3. **连接**：
   - **主机**：您的 NAS IP
   - **端口**：您设置的端口
   - **用户名**：`root`
   - **密码**：您设置的密码

## 开发与集成

如果您是开发者，可以通过以下方式与本应用交互：

- **服务控制**：应用中心会自动调用 `cmd/main` 脚本进行管理。
- **自定义配置**：修改 `/usr/local/apps/@appcenter/MariaDB12/etc/my.cnf` 后，建议通过应用中心的“设置”界面提交以触发配置重载。
- **备份与恢复**：建议定期备份 `var/data` 目录或使用标准的 `mysqldump` 工具。
