**WebServer** 是一款专为飞牛 NAS 系统 (fnOS) 深度定制的轻量级可视化 Web 环境配置与管理工具。它旨在通过直观的 Web 界面，帮助用户轻松管理 Nginx、PHP、反向代理、数据库以及网站站点，彻底告别繁琐的命令行操作。

---

## 🌟 核心特性

- **🚀 极简站点管理**
  - 支持 **端口模式** (IP:Port) 与 **域名模式** (Domain) 快速创建站点。
  - 一键开启/停用站点，内置 Nginx 配置语法自动检测，确保服务稳定。
  - 内置**权限修复工具**，一键解决常见的 403 Forbidden 权限问题。

- **🔄 高级反向代理 (New)**
  - 基于高性能 Go 引擎实现，独立于 Nginx 运行，互不干扰。
  - **自动证书匹配**：深度集成系统证书管理，自动发现并应用 HTTPS 证书。
  - 支持通配符域名匹配及多种后端协议（HTTP/HTTPS）转发。
  - 提供单页可视化规则管理，秒级生效。

- **🎛️ 多版本 PHP 全兼容**
  - 自动检测系统中已安装的所有 PHP-FPM 版本。
  - 支持为每个站点独立指定 PHP 版本，轻松实现多版本共存。
  - 实时监控 PHP 服务运行状态。

- **🌐 Nginx 深度集成**
  - 提供 Nginx 服务状态监控、平滑重启等维护操作。
  - 全局管理上传限制 (`client_max_body_size`)，统一调整站点的文件吞吐能力。
  - 支持自定义 **伪静态 (Rewrite)** 规则。

- **🗄️ 数据库环境管理**
  - **原生服务监控**：实时检测系统中安装的 MariaDB、MySQL、PostgreSQL 服务。
  - **Docker 实例管理**：支持一键部署隔离的 Docker 版数据库，集成 phpMyAdmin 面板。
  - **实例 ID 隔离**：确保面板操作不会干扰用户自行部署的 Docker 容器。

---

## 🛠️ 技术架构

- **前端交互**：基于 HTML5, Vanilla CSS 以及 Layui 框架，提供精美的现代化响应式界面。
- **混合后端控制**：
  - **站点管理**：采用 Shell (CGI) 脚本直接与系统 Nginx 协同工作，资源占用极低。
  - **反向代理**：内置专用 Go 后端引擎，通过 Unix Socket 进行内部高效通信。
- **安全性**：所有配置变更均经过语法合法性检测，并支持配置回滚。

---

## 📂 项目结构 (Developer)

为了保持工程的整洁，本项目采用了 **源码与发行包分离** 的结构：

- `app/`: 待打包的发行程序包，包含二进制执行文件、UI 资源及 CGI 脚本。
- `src/`: 后端 Go 程序的源代码，仅供开发和编译使用，不包含在安装包内。
- `cmd/main`: 负责管理整个 WebServer 应用生命周期的启动/关闭脚本。
- `fnpack.exe`: fnOS 专用的打包工具。

## 📦 安装与使用

1. **环境要求**：
   - 运行环境：飞牛 NAS 系统 (fnOS)。
   - 依赖项：建议预先安装 `nginx` 和所需版本的 `php-fpm`。

2. **快速上手**：
   - 进入 WebServer 应用，在“概览”页确认环境状态。
   - **创建站点**：在“网站管理”中按向导创建 Nginx 站点。
   - **配置反代**：在“反向代理”中添加映射规则，方便访问 Docker 容器或内网服务。
